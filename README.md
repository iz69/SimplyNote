自鯖にsimplenoteぽいのを入れたくて自分用に作ったもの<br/>
データの保管先は自鯖(APIサーバ) or Google Drive<br/>
気が向いたときに更新。永遠に開発中。

I wanted a Simplenote-like app on my own server, so I built this for personal use.<br/>
Notes can be stored either on my self-hosted API server or on Google Drive.<br/>
I update it whenever I feel like it—forever under development.<br/>

- debian + nginx (proxy_pass)
- docker
 - simplynote
 - simplynote-api

Other client
- simplynote-app (android apps : https://play.google.com/store/apps/details?id=com.kuromaru_fx.simplynote)
- simplynote-vim (vim plugin : https://github.com/iz69/SimplyNote_vim) ← 最近あまり使ってない
